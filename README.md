# vMix Patches

Community-driven enhancement patches for vMix Live Production Software.

This repository provides **automation scripts, workflow patches, overlays, and performance optimizations**
built using **official vMix APIs and supported features**.

## Features
- Audio & video workflow improvements
- Live streaming automation
- Dynamic title & overlay patches
- Companion & scripting support
- Performance optimization guides

## Supported vMix Versions
- vMix 29.0.0.41
- vMix 29

## Disclaimer
This project does NOT modify vMix binaries or bypass licensing.
All patches use official APIs, shortcuts, or supported integrations.
